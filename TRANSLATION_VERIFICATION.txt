HOW TO USE
================================================================================

1. CLIENT COMPONENT:

   import { useTranslation } from "@/components/LanguageProvider";
   
   export default function MyComponent() {
     const { t } = useTranslation();
     return <h1>{t("sidebar.dashboard")}</h1>;
   }

2. SERVER COMPONENT:
   - Create client wrapper component
   - Import data in server component
   - Pass data to client component
   - Client component renders with translations

   - Selects English or Español
   - Change is instant and persists

SYSTEM ARCHITECTURE
================================================================================

Translation Flow:
1. User visits app
2. App loads language from localStorage (instant)
3. LanguageProvider context makes it available
4. useTranslation() hook provides t() function
5. Components use t("key") for all strings
6. Language change updates localStorage + API
7. Page reload restores from localStorage

Component Tree:
App
├── Providers (Updated)
